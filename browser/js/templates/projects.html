Cohearence

Cohearence is a Chrome extension that automatically remembers what you listen to on YouTube, Spotify, Soundcloud, and Bandcamp, giving you one central database to listen to and manage music you find online.

Technologies used:

  • Node.js
  • Angular.js
  • Express.js
  • MongoDB
  • Echo Nest API
  • Youtube Search API
  • Youtube IFrame Player API

Spheres

A web-based musical memory game, similar to Simon, in which players repeat increasingly-complicated note patterns in order to build constellations in the sky.

Proud Winner of Best Design Award at Fullstack Academy 1507 Hackathon

Technologies used:

  • Node.js
  • Angular.js
  • Express.js
  • SVG Animations
  • CSS Animations
  • Web Audio API

I Don't Care, Just Give Me Things

An e-commerce site built with the MEAN Stack where users can create and design their own storefronts.

Every 10 seconds, a new random item in the store goes on sale for 100% off, but you have find it.

Technologies used:

  • Node.js
  • Angular.js
  • Express.js
  • MongoDB
  • Sockets
